TROUBLESHOOTING
================================================================================

"Gemini API key not configured"
    Solution: Create ML/.env with GEMINI_API_KEY=your_key_here
    Get key: https://makersuite.google.com/app/apikey

"No signals generated"
    This is NORMAL! System is selective.
    Means: No high-confidence setups found
    Solution: Try different universe or scan later

"Port 5001 already in use"
    Solution: Change port in web_app/app.py:
    app.run(debug=True, host='0.0.0.0', port=5002)

"Charts not loading"
    Solution: Check internet (Plotly uses CDN)
    Disable ad blockers

For more: See WEB_UI_GUIDE.md -> Troubleshooting section
